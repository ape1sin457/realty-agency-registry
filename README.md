# Реєстр квартир агентства нерухомості
Курсовий проєкт з дисципліни "Технології створення програмних продуктів".

## Опис
Програма призначена для автоматизації обліку об'єктів нерухомості. 
Дозволяє додавати нові квартири, зберігати їх у базу даних та переглядати список.

## Стек технологій
* Python 3.x
* SQLite3 (База даних)
* Tkinter (Графічний інтерфейс)

## Як запустити
1. Встановіть Python.
2. Запустіть файл: `python main.py`
